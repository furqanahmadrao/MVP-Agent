
### What is the MVP Agent?

The MVP Agent is a multi-agent system that transforms simple startup ideas into professional Product Requirements Documents (PRDs) and technical specifications.

### How does it work?

The MVP Agent uses a team of specialized AI agents to conduct market research, design architecture, and plan sprints. The agents work together in a LangGraph workflow to generate a comprehensive blueprint for your project.

### What is the BMAD methodology?

The BMAD (Breakthrough Method for Agile AI-Driven Development) methodology is a four-phase process that mirrors the structure of a real-world product team. The phases are Analysis, Planning, Solutioning, and Implementation.

### What is LangGraph?

LangGraph is a library for building stateful, multi-agent applications with LLMs. It is used to orchestrate the workflow of the MVP Agent.

### What is TOON?

TOON (Token-Oriented Object Notation) is a format that can reduce token usage by 30-60%. This makes it more affordable to generate large, complex documents.

### Is it free?

The MVP Agent is open-source and free to use. However, you will need to provide your own Google Gemini API key to use the application.
