# Troubleshooting

This guide provides solutions to common problems that you may encounter while using the MVP Agent.

## "Invalid API Key" Error

This error occurs when the **Google Gemini API Key** is missing or invalid. To resolve this issue, please do the following:

1.  Navigate to the **Settings** tab in the UI.
2.  Make sure that you have entered a valid API key in the **Google Gemini API Key** text field.
3.  Click **Save**.

## "Docker Container Not Starting" Error

This error can occur for a variety of reasons. Here are some things to check:

-   Make sure that you have Docker installed and running.
-   Make sure that you have built the Docker image successfully.
-   Make sure that port 7860 is not already in use.

## "Generated Documents are Low Quality" Error

The quality of the generated documents is highly dependent on the quality of the input. To ensure the best results, please do the following:

-   Be as specific as possible when describing your startup idea.
-   Provide as much context as possible.
-   Use clear and concise language.
